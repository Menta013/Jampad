# Final Fusion exports required

The included `Jampad_PCB_Assembly.step` is the KiCad mechanical PCB export.
It is not the required complete assembled hackpad CAD file.

Before submission, export these files from the finished Fusion 360 design:

1. `Jampad_Complete_Assembly.step`
   - Must contain the PCB/electronics, top case, and bottom case together.
2. `Top.step`
   - Export only the printable top case component.
3. `Bottom.step`
   - Export only the printable bottom case component.

Place the complete assembly in this `CAD/` folder. Place `Top.step` and
`Bottom.step` in the project-level `production/` folder.

After exporting, replace the assembly screenshot in `assets/` if a cleaner,
final view is available.
